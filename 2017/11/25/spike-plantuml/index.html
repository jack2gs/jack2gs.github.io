1.2 声明参与者(Declaring participant)

1.2.1 要点

  • 关键字 participant 用于改变参与者的顺序。此外还有其他的关键字可以声明参与者：

    名称 用途
    actor 参与者
    boundary 边界
    control 控制
    entity 实体
    database 数据库
  • 关键字 as 用于重命名参与者
  • 可以自定义各种参与者的背景颜色

1.2.2 总结

  • 自定义参与者，并更改显示顺序 participant, actor, boundary, control, entity, database 和 =collections=用于声明不同的参与者
  • 使用 as 重命名参与者
  • 自定义颜色 可以使用 RGB 和颜色英文名来指定颜色。
  • 注释
    • 使用 ' 定义单行注释
    • 使用 /'...'/ 定义多行注释
